# Universal Builder Core

Este directorio contiene el núcleo de la extensión Universal Builder para SketchUp.

## Archivos principales

- extension.rb
- loader.rb
- version.rb

## Responsabilidades

- Registrar la extensión en SketchUp.
- Cargar todos los módulos del proyecto.
- Administrar la versión.
- Inicializar Universal Builder de forma segura.

Todo el sistema comienza desde este módulo.
